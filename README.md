# Rule-Based Chatbot

## CodeAlpha Internship Task

This project is a Rule-Based Chatbot developed using Python as part of the CodeAlpha Internship Program.

## Project Description

The Rule-Based Chatbot is designed to interact with users through predefined rules and responses. It analyzes user input and generates appropriate replies based on specific keywords and conditions. The project demonstrates the implementation of basic conversational AI concepts using Python.

## Features

* Interactive conversation with users
* Keyword-based response generation
* Predefined rules and conditions
* Simple and user-friendly interface
* Demonstrates basic chatbot functionality

## Technologies Used

* Python

## How to Run

1. Clone or download the repository.
2. Open the project folder.
3. Run the following command:

python rule_based_chatbot.py
